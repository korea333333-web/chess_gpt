# GAMBIT (갬빗) 테스트 기준

각 스프린트는 아래 기준을 **모두 통과**해야 완료로 판정한다.
기준은 "잘 된다" 같은 모호한 표현이 아니라 **검증 가능한 형태**로 적는다.

---

## Sprint 1 완료 기준 — 체스 두뇌 (코어 엔진)

### TC-1.1: 게임 상태 관리
- [ ] 시작 시 표준 포지션이 정확히 생성된다 (FEN: `rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1`)
- [ ] 합법 수를 적용하면 게임 상태(보드·차례)가 정확히 갱신된다
- [ ] 현재 차례(백/흑)를 항상 정확히 추적한다

### TC-1.2: 규칙 정확성
- [ ] 임의 포지션의 합법 수 목록이 chess.js 기준과 100% 일치한다
- [ ] 비합법 수(예: 폰을 옆으로, 막힌 길로 룩 이동)는 거부된다
- [ ] 캐슬링(킹·퀸 사이드)이 조건에 맞을 때만 허용된다
- [ ] 앙파상이 정확한 타이밍에만 허용된다
- [ ] 폰이 끝줄 도달 시 승급(퀸/룩/비숍/나이트 선택)이 동작한다

### TC-1.3: 종료 판정
- [ ] 바보 메이트(`1.f3 e5 2.g4 Qh4#`) 입력 시 **체크메이트**로 판정한다
- [ ] 대표 스테일메이트 포지션에서 **스테일메이트(무승부)**로 판정한다
- [ ] 50수 규칙, 동형 3회 반복 시 무승부로 판정한다
- [ ] 체크 상태를 정확히 감지한다

### TC-1.4: 기보 & 무르기
- [ ] 둔 수가 표준 PGN으로 기록된다
- [ ] 무르기(undo) 시 직전 상태로 정확히 복원된다(여러 번 연속 포함)

### TC-1.5: 프로젝트 셋업 & 배포 파이프라인
- [ ] `npm run dev`로 로컬 실행된다(빈 화면이라도)
- [ ] `npm run build`가 에러 없이 성공한다
- [ ] GitHub 푸시 시 Vercel 빌드·배포가 성공한다

### TC-1.6: 자동화 테스트
- [ ] 위 규칙(TC-1.1~1.4)을 검증하는 jest 단위 테스트가 작성되어 있다
- [ ] `npm test` 실행 시 모든 테스트가 통과한다

## Sprint 2 완료 기준 — 입체 체스판 (3D)
> 스프린트 시작 시 세부화. 핵심 골격:
- [ ] 3D 체스판이 버드아이뷰(비스듬한 탑다운)로 화면에 표시된다
- [ ] 마우스 드래그로 시점 회전, 휠로 줌이 되며 과한 각도는 제한된다
- [ ] 6종 말이 모두 식별 가능한 형태로 보인다(나이트 포함)
- [ ] Sprint 1 상태와 연결되어 32개 말이 정확한 칸에 배치된다
- [ ] 데스크톱에서 끊김 없이(체감 60fps 수준) 렌더된다

## Sprint 3 완료 기준 — 플레이 가능
> 스프린트 시작 시 세부화. 핵심 골격:
- [ ] 말 클릭 시 갈 수 있는 칸이 하이라이트되고, 클릭으로 이동된다
- [ ] 비합법 이동은 화면에서도 막힌다(Sprint1 로직과 일치)
- [ ] 핫시트 모드로 사람 둘이 끝까지 한 판을 둘 수 있다
- [ ] vs 컴퓨터 모드에서 AI가 합법 수로 응수하며, 난이도 3단계가 동작한다
- [ ] 체크메이트/무승부 시 결과가 화면에 표시된다
- [ ] 새 게임·무르기·항복 버튼이 동작한다

## Sprint 4 완료 기준 — 실사 비주얼
> 스프린트 시작 시 세부화. 핵심 골격:
- [ ] 말·보드에 PBR 재질과 HDRI 반사가 적용되어 "고급 제품 사진" 인상을 준다
- [ ] 부드러운 그림자와 후처리(AO/톤매핑)가 적용된다
- [ ] 말 이동/잡기 연출과 효과음이 자연스럽다
- [ ] 모바일에서 레이아웃이 깨지지 않고 30fps 이상 유지된다
- [ ] 시작/결과 화면이 통일된 디자인으로 완성된다
- [ ] (평가) 비주얼 완성도 자체 점검 7/10 이상

## Sprint 5 완료 기준 — 배포 & 마무리
> 스프린트 시작 시 세부화. 핵심 골격:
- [ ] 실제 Vercel URL로 접속해 처음부터 끝까지 플레이된다
- [ ] 주요 브라우저(크롬·사파리·엣지)와 모바일에서 정상 동작한다
- [ ] 초기 로딩 3초 이내(에셋 지연 로딩 적용)
- [ ] README와 2단계(온라인) 설계 메모가 작성되어 있다
